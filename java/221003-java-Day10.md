## 느낀점

못들었던 강의를 빠르게 들었다. 자바의 자료구조에 대해 공부했는데 생각보다 많은 라이브러리들이 있길래 활용만 잘하면 구현할 수 있는 것들이 많을 것 같다는 생각이 들었다. 제네릭 프로그래밍을 공부하면서도 자바는 정말 코드 재사용 효율이 좋은 언어인 것 같다고 느꼈다.
